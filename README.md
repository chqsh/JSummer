# JSummer
JSummer is a tool to compute and check MD5, MD4, MD2, SHA-160, SHA-256, SHA-384, SHA-512, RIPEMD128, RIPEMD160, WHIRLPOOL, TIGER, HAVAL message digest consisting of a Console-Version and GUI. Implemented in Java.

Original website: https://sourceforge.net/projects/jsummer/

JSummer-0.1.0.0-java.msi for java x86 (swt for win32) for Windows

The md5 file must is ANSI / GBK codepage. JSummer do NOT support UTF-8 md5 file.
